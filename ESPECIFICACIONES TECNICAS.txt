MODULOS DE RECEPCION Y CONTROL DE MEDICAMENTOS

	-MOTOR DE BASE DE DATOS: MYSQL
	-SERVIDOR DE BASE DE DATOS: MARIA DB V10.4.24
	-SERVIDOR WEB: APACHE/2.4.53
	-VISUAL STUDIO CODE.
	-GIT
	-GITHUB
	-SERVIDOR DE APLICACIONES XAMPP V3.3.0
	-TECNOLOGIA DE DESARROLLO PHP.
	-HTML
	-JAVASCRIPT
	-JQUERY
	-DATA TABLES
	-BOOTSTRAPS

3)	PARA LA INSTALACION DEL MODULO SE NECESITA UN SERVIDOR DE APLICACIONES COMO EL ANTES MENCIONADO (XAMPP), AL EJECUTAR EL SERVIDOR SE DIRIGE AL NAVEGADOR Y EN LA BARRA DE NAVEGACION
	SE ESCRIBE LA DIRECCION 12.0.0.1/PHPMYADMIN EN EL CUAL ABRIRA EL MOTOR DE LA BASE DE DATOS DONDE ADJUNTAREMOS (IMPORTAR) EL OBJETO DE BASE DE DATOS UBICADO DENTRO DE LA CARPETA MODEL DE NUESTRO PROYECTO
	LUEGO DE ADJUNTAR EL ARCHIVO DE LA BASE DE DATOS NOS DIRIGIMOS NUEVAMENTE AL NAVEGADOR Y EN LA BARRA DE NAVEGACION SE ESCRIBE LA DIRECCION  127.0.0.1/Modulo_Web_Medicamentos.
	EN LA CUAL SE DESPLEGARA LOS MODULOS DESARROLLADOS.
